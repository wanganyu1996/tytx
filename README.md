# tytx
大二用PHP写的统一通信网站
统一统一通信网站主要实现登录，通话记录的查询、CSV格式导出、修改、添加、试听、CSV格式导入等功能
其中查询包括按当天、昨天、过去一周、最近一个月，某一天，自定义起止时间等条件查询。
